## stringCat

Write a pointer version of the function  `strncat`  that `strncat(s,t, n)`  copies the first `n` characters of string  `t`  to the end of  `s`  .


### Example 

**input** 
abc de 2
**output** 
abcde

**input** 
hello world 3 
**output** 
hellowor
