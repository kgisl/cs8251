## stringEnd 

Write the function `strend(const char *s, const char *t)`, which returns 1 if the string `t` occurs at the end of the string `s`, and zero otherwise.

### Example 

**input**
		
		abcde  de  

**output** 
1 

**input**
`abcde` `abcde`  
**output** 
1 

**input**
`abcde` `dz`  
**output** 
0
